Git & GitHub

Git is a version control system that allows you to track changes in your code and collaborate with others.
GitHub is a platform that hosts Git repositories and provides tools for collaboration.

Key Concepts

  • Repository: A collection of files and their history.
  • Commit: A snapshot of your repository at a point in time.
  • Branch: A parallel version of your repository.
  • Merge: Combining changes from different branches.
  • Clone: Combining changes from different branches.
